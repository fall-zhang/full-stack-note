## Win 连接 Mac

- 在 Mac 上，选取苹果菜单 <img src="https://help.apple.com/assets/68CC94A874C2BF154000B0D2/68CC94AAFCF82C5E1D0E5465/zh_CN/2f77cc85238452e25cb517130188bf99.png" alt="img"  /> “系统设置”，点按边栏中的“通用” <img src="https://help.apple.com/assets/68CC94A874C2BF154000B0D2/68CC94AAFCF82C5E1D0E5465/zh_CN/7c4faef8373d3d73052729c76d485046.png" alt="img"  />，然后点按“共享”。（你可能需要向下滚动。）
- 点按“远程登录”旁边的 <img src="https://help.apple.com/assets/68CC94A874C2BF154000B0D2/68CC94AAFCF82C5E1D0E5465/zh_CN/974db3c051f6ac0adaddf1e2fec1001e.png" alt="“信息”按钮" />。
- 打开“远程登录”。
- 若要让远程用户可完全访问 Mac 上的磁盘，请打开“允许远程用户对磁盘进行完全访问”。

vscode 配置

```json
{
  // 添加这两项，避免多次重新连接
  "remote.SSH.useLocalServer": false,
  "remote.SSH.useExecServer": false,
}
```

### 意外情况

如果退出的时候未关闭，可能会导致多次重新连接

- 使用 vscode 控制台的命令  `Kill VS Code Setver on Host` 然后重新连接
- 退出时使用该命令：`Close Remote Connection`
  - 或者点击左下角的服务器 ip 给出的配置


删除 vscode 服务器 token

```bash
sudo rm -rf /home/<user>/.vscode-server/bin/<token>
```

### 额外配置

```json
{
  // 本地下载 server 并传输到目标主机
  "remote.SSH.localServerDownload": "always",
}
```
