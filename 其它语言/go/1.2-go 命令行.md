## Go

安装 go，会自带一个命令行工具

go 不同于 node，是将文件编译为二进制文件去跑在不同的系统上，你需要设置环境变量 `GOOS` 以及 `GOARCH`

### 命令行

常用命令

```bash
# 获取命令的详情
go help <command>
# 初始化 go 项目
go init
# 运行 go 项目
go run <filename.go>
# 打包 go 项目
go build
# 格式化代码
go fmt
# 检查代码常见错误以及问题
go vet
```
