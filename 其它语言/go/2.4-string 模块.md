## strings

### 分割字符串

```go
import (
  "fmt"
  "strings"
)
func SplitString() {
  str := "Go-语言-欢迎您"
  parts := strings.Split(str, "-")
  fmt.Println(parts) // 输出: ["Go", "语言", "欢迎您"]
}

func SplitStrFun(){
  str := "Go,语言,欢迎您"
  parts := strings.FieldsFunc(str, func(r rune) bool {
  	return r == ','
  })
}

```
